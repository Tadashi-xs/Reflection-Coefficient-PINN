# Reflection Coefficient PINN

## Структура проекта

```text
.
├── generate_data.py              # генерация синтетического датасета
├── requirements.txt              # зависимости проекта
├── storage/                      # сгенерированные датасеты
├── visualization/                # сохраненные графики и метрики
└── PINN/
    ├── PhysicsInformedNN.ipynb   # основной ноутбук обучения PINN
    └── saved_models/             # сохраненные обученные модели
```

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Генерация датасета

Базовый запуск:

```bash
python generate_data.py
```

По умолчанию будет создан датасет на `10000` строк:

```text
storage/dataset_10k.csv
```

Чтобы задать размер выборки вручную:

```bash
python generate_data.py --num_samples 30000
```

Такой запуск создаст файл

```text
storage/dataset_30k.csv
```

Количество объектов должно быть положительным и кратным `1000`, потому что имя файла по умолчанию строится в формате:

```text
storage/dataset_{количество_тысяч}k.csv
```

Если нужно сохранить датасет по конкретному пути:

```bash
python generate_data.py --num_samples 30000 --output storage/custom_dataset.csv
```

## Основные параметры генерации

### Размер датасета и диапазон волнового числа

| Параметр | Значение по умолчанию | Смысл |
|---|---:|---|
| `--num_samples` | `10000` | количество строк в датасете; должно быть кратно `1000` |
| `--output` | `None` | путь сохранения `.csv`; если не указан, строится автоматически |
| `--kappa_min` | `1.0e-4` | минимальное значение волнового числа `kappa` |
| `--kappa_max` | `20.0` | максимальное значение волнового числа `kappa` |
| `--chunk_size` | `5000` | размер блока при вычислении датасета |
| `--logit_eps` | `1.0e-6` | ограничение для вычисления `logit_R` |

### Физические параметры пластины

| Параметр | Значение по умолчанию | Смысл |
|---|---:|---|
| `--rho` | `7800.0` | плотность материала пластины |
| `--thickness` | `0.002` | толщина пластины |
| `--young_modulus` | `2.0e11` | модуль Юнга |
| `--poisson_ratio` | `0.3` | коэффициент Пуассона |

### Параметры решетки и резонатора

| Параметр | Значение по умолчанию | Смысл |
|---|---:|---|
| `--period` | `0.05` | период массива резонаторов |
| `--n_terms` | `80` | число членов суммы в каждую сторону: `-n_terms, ..., n_terms` |
| `--omega0` | `2π · 100` | собственная частота резонатора |
| `--damping_ratio` | `0.01` | коэффициент демпфирования |
| `--mu_scale` | `1.0` | масштаб параметра взаимодействия резонатора с пластиной |

### Геометрия двух массивов резонаторов

| Параметр | Значение по умолчанию | Смысл |
|---|---:|---|
| `--geometry` | `zigzag` | способ задания смещения второго массива: `zigzag`, `half_period` или `custom` |
| `--fixed_psi` | `π / 6` | фиксированный угол падения волны |
| `--fixed_dx_fraction` | `0.5` | доля периода для `delta_x` при `geometry=custom` |
| `--fixed_dy_fraction` | `0.5 * tan(π / 6)` | доля периода для `delta_y` при `geometry=custom` |

## Содержание датасета

В итоговый `.csv` сохраняются следующие столбцы:

```text
kappa, omega, psi, delta_x, delta_y, R, logit_R
```

Смысл столбцов:

| Столбец | Смысл |
|---|---|
| `kappa` | волновое число; основной аргумент сетки |
| `omega` | циклическая частота, вычисленная из `kappa` и параметров пластины |
| `psi` | фиксированный угол падения волны |
| `delta_x` | горизонтальный сдвиг второго массива резонаторов |
| `delta_y` | вертикальный сдвиг второго массива резонаторов |
| `R` | коэффициент отражения, обрезанный в диапазон `[0, 1]` |
| `logit_R` | логит-преобразование `R` |

## Как запускать обучение

1. Установить зависимости:

```bash
pip install -r requirements.txt
```

2. Сгенерировать датасет нужного размера:

```bash
python generate_data.py --num_samples 30000
```

3. Открыть ноутбук:

```text
PINN/PhysicsInformedNN.ipynb
```

4. В `Config` ноутбука указать путь к датасету:

```python
data_path = project_root / "storage" / "dataset_30k.csv"
```

5. Запустить ноутбук.

После обучения модели сохраняются в `PINN/saved_models/`, а графики и метрики — в `visualization/`.
